## Interaction

| Command                                      | Description                                |
| -------------------------------------------- | ------------------------------------------ |
| `agent-computer-use click [selector] -a App` | Click an element (AXPress, no focus steal) |
| `agent-computer-use type "text" -s @e3`      | Clear and type into a field (AXSetValue)   |
| `agent-computer-use type "text" -a App`      | Type into focused field (keyboard sim)     |
| `agent-computer-use key cmd+k -a Slack`      | Press a key or combo                       |
| `agent-computer-use scroll down -a Music`    | Scroll within an app                       |
| `agent-computer-use scroll-to @e42`          | Scroll element into view (headless)        |
| `agent-computer-use drag @e5 @e10 -a Finder` | Drag from one element to another           |

## Snapshot Options

```bash
agent-computer-use snapshot -a Calculator              # full tree
agent-computer-use snapshot -a Calculator -i           # interactive elements only
agent-computer-use snapshot -a Calculator -c           # compact (skip empty containers)
agent-computer-use snapshot -a Calculator -d 3         # limit depth
agent-computer-use snapshot -a Calculator -i -c -d 5   # combine options
```

| Option              | Description                                        |
| ------------------- | -------------------------------------------------- |
| `-a, --app`         | Target application                                 |
| `-i, --interactive` | Only interactive elements (buttons, inputs, links) |
| `-c, --compact`     | Remove empty structural elements                   |
| `-d, --depth`       | Limit tree depth                                   |

## Global Options

| Option              | Description                            |
| ------------------- | -------------------------------------- |
| `--human`           | Human-readable output                  |
| `--compact`         | Compact single-line JSON               |
| `--timeout <s>`     | Global timeout in seconds (default: 5) |
| `-v, --verbose`     | Debug traces                           |
| `--cdp`             | Force CDP mode for Electron apps       |
| `--cdp-port <port>` | Override CDP port                      |
| `--no-cdp`          | Disable CDP auto-detection             |

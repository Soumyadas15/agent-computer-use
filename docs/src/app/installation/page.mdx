## Quick setup (recommended)

```bash
git clone https://github.com/kortix-ai/agent-computer-use
cd agent-computer-use
./scripts/setup.sh
```

The setup script:

1. Installs Rust (if needed)
2. Installs dependencies
3. Builds and installs `agent-computer-use` to `~/.cargo/bin`
4. Prompts for accessibility permissions

## Cargo

```bash
cargo install --git https://github.com/kortix-ai/agent-computer-use --path cli
```

## From source

```bash
git clone https://github.com/kortix-ai/agent-computer-use
cd agent-computer-use/cli
cargo build --release
cargo install --path .
```

## Permissions

agent-computer-use needs accessibility access to control apps:

1. Open **System Settings → Privacy & Security → Accessibility**
2. Add your terminal app (Terminal, iTerm2, Warp, Cursor, VS Code)
3. Toggle it on

Verify:

```bash
agent-computer-use check-permissions
# → Accessibility permissions: granted
```

## Verify installation

```bash
agent-computer-use --version
agent-computer-use apps           # should list running applications
```

## Uninstall

If you installed via Cargo (setup script, `cargo install`, or from source):

```bash
cargo uninstall agent-computer-use
```

If you installed via a Node package manager:
